## HOW THE 5 ENTITIES REINFORCE EACH OTHER

### Loop 1: Evidence → Publication → Distribution

```
Nama CRO runs trial → produces structured clinical data
    → Waraqa publishes results in peer-reviewed journal
    → Evidence Platform indexes into knowledge graph
    → Physicians access at point of care
    → Evidence gaps detected → new research priorities
    → Nama CRO designs next trial → [LOOP REPEATS]
```

**Compound effect:** Each loop adds knowledge nodes, publications, and physician engagement — the graph grows exponentially.

### Loop 2: Training → Distribution → Research

```
Nama Academy trains 2,000+ physicians/year
    → Each receives Evidence Platform account ($0 CAC)
    → Physicians use platform for clinical decisions
    → Platform captures questions and reveals evidence gaps
    → Evidence gaps become Nama CRO research priorities
    → Trial results feed Waraqa publications
    → New publications become Nama Academy curriculum
    → [LOOP REPEATS with larger physician base]
```

**Compound effect:** Physician network grows linearly while platform value grows exponentially.

### Loop 3: Drug Localization → Publication → Intelligence

```
Nama Bio runs BE study → produces Saudi PK/PD data
    → Waraqa publishes results
    → Evidence Platform adds drug data to knowledge graph
    → CII uses data for prescribing intelligence
    → Physicians make better prescribing decisions
    → More drugs localized → more data → [LOOP REPEATS]
```

**Compound effect:** Saudi population pharmacokinetic data is irreplaceable — each study makes the dataset more valuable.

### Loop 4: Society Partnerships → Multi-Entity Revenue

```
Waraqa partners with scientific society
    → Launches society journal (40-60% revenue share)
    → Nama Academy co-organizes society CME (40-50% revenue share)
    → Evidence Platform hosts society digital hub
    → Nama CRO recruits society experts as PIs
    → Society reaches Category A SCFHS rating
    → More societies see the model → more partnerships
    → [LOOP REPEATS across 80+ societies]
```

**Compound effect:** Each society partnership generates revenue for 4 TKG entities simultaneously.

### Loop 5: PubHelper — Clinical Questions → Scientific Assets (THE RESEARCH FACTORY)

```
Welunion clinician asks a clinical question
    → PubHelper Intake captures and structures (PICO)
    → AI assigns evidence level from pyramid (Level 1-5)
    → Project Designer generates protocol
    → Data captured from Welunion/CRO/external (eCRFs)
    → Analysis Workbench produces results
    → Writing Studio drafts manuscript (using CII NLP engine)
    → Publication Manager routes to Waraqa journal
    → Evidence Engine grades output (SOR: A/B/C)
    → Graded evidence enters Evidence Platform knowledge graph
    → CII clinical agents update decision support
    → Clinicians receive better answers → ask better questions
    → [LOOP REPEATS — EACH QUESTION BECOMES A KNOWLEDGE ASSET]
```

**Compound effect:** PubHelper converts the most abundant raw material in healthcare (clinical questions) into the most valuable refined product (graded evidence). Every question answered makes the Evidence Platform smarter, every study published strengthens Waraqa's credibility, and every Grade A output justifies a VBH Co value-based contract.

**Why PubHelper is the flywheel accelerator:** Without PubHelper, research initiation depends on individual researcher motivation. With PubHelper, the pipeline from question to evidence is **automated, AI-augmented, and systematic** — turning an opportunistic process into a production line.
