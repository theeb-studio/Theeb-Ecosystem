## SCFHS Integration: The Strategic Opportunity

**Current state:** SCFHS provides UpToDate to Saudi clinicians at significant cost. Utilization rates are low because UpToDate is not Saudi-specific, not Arabic-optimized, and not integrated with CME tracking.

**Saudi Evidence proposition to SCFHS:**

| Dimension | UpToDate (Current) | Saudi Evidence (Proposed) |
|-----------|-------------------|--------------------------|
| Cost to SCFHS | SAR 20M+/year | Free (V1) or significantly lower |
| Language | English only | Arabic + English |
| Saudi content | None | Saudi guidelines, local evidence, MNC criteria |
| CME integration | None | Built-in Learning Passport with auto-credit |
| Utilization | ~15-30% of licensed professionals | Target 50%+ (free, mobile, Arabic) |
| Local evidence | Zero | PubHelper research pipeline feeds continuously |
| Society integration | None | 80+ society hubs with guidelines |
| Research connection | None | PubHelper intake directly from search |
| Data value | None to SCFHS | Workforce intelligence: knowledge gaps, training needs, competency |

**The ask to SCFHS (V2):**
1. Redirect portion of UpToDate budget to Saudi Evidence development
2. Integrate Learning Passport with SCFHS CME tracking system (SSO + API)
3. Endorse Saudi Evidence as a recommended resource for licensed professionals
4. Provide SCFHS registry data for user verification (license number validation)

## Stakeholder Journey Through the Ecosystem

Every stakeholder that enters through Saudi Evidence generates value for multiple Theeb entities:

```

    │
    ├──► Uses evidence search → data for CII training
    ├──► Earns CME credits → tracked in Learning Passport
    ├──► Submits research question → PubHelper project
    ├──► Subscribes to Physician Reputation → Waraqa content

    ├──► Buys MGA bundle → Nexus MGA revenue
    ├──► Recommends to hospital → institutional deployment
    │
    Hospital deploys Saudi Evidence institutionally
    │
    ├──► CBAHI compliance tool → institutional revenue
    ├──► CDI training need identified → Nama Academy CDI program
    ├──► Revenue cycle gaps surfaced → Integra RCM engagement
    ├──► Research activity tracked → PubHelper institutional account
    ├──► Staff training tracked → Nama Academy AaaS contract
    │
    Scientific society joins
    │
    ├──► Journal launched → Waraqa revenue
    ├──► CME delivered → Nama Academy revenue
    ├──► Guidelines hosted → Saudi Evidence content
    ├──► Members join platform → user growth
    ├──► Research coordinated → PubHelper + Nama CRO
```

**Every stakeholder is a multiplier. No stakeholder is a dead end.**
