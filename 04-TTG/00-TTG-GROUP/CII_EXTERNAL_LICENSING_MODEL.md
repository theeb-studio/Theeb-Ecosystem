## 4. COMPETITIVE PROTECTION

| Protection | Mechanism |
|-----------|-----------|
| **Ecosystem data advantage** | CII models trained on Welunion + Integra RCM + Nama CRO data — external clients get the intelligence but NOT the raw data |
| **Pricing leverage** | Internal entities get cost-recovery pricing; external clients pay market rates (3-5× markup) |
| **Feature sequencing** | Ecosystem entities always get new CII capabilities 6-12 months before external clients |
| **Data sovereignty** | External client data processed in sovereign cloud but NEVER mixed with ecosystem data |
