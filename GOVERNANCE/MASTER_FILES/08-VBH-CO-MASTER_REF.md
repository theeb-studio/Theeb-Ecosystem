## CORE MISSION

Measure what healthcare actually delivers — building the outcomes measurement, cost analytics, and contract design infrastructure that enables Saudi Arabia's transition from volume-based to value-based healthcare payment.

---

## STRATEGIC MOAT

| Moat Layer | Description |
|-----------|-------------|
| **PROMs Infrastructure** | First-mover on Patient-Reported Outcome Measures collection at scale in KSA |
| **Cost-of-Care Analytics** | Proprietary cost modeling using real clinical-financial data from Integra RCM |
| **VBP Contract Design** | Only company in KSA building bundled payment engines and shared savings contracts |
| **AR-DRG Intelligence** | Deep analytics on the new hospital payment system — advisory moat |
| **Payer-Provider Bridge** | Uniquely positioned between providers (via Integra RCM) and payers (via Nexus MGA) |
