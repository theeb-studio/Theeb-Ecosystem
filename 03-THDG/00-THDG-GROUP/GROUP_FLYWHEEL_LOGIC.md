## THE THDG FLYWHEEL

THDG operates as the **point-of-care data generation engine** — the place where the Theeb Ecosystem touches patients. Every clinical encounter, prescription, and outcome generates data that powers TVG's financial analytics, TKG's research, and TTG's artificial intelligence. The flywheel: more clinics → more encounters → more data → smarter AI → better care → more patients → more clinics.

## HOW THE ENTITIES REINFORCE EACH OTHER

### Loop 1: Clinic → Pharmacy → Revenue Capture

```
Welunion clinic sees patient
    → Physician prescribes medication
    → TAP Pharmacy dispenses in-clinic (revenue capture)
    → Prescription data flows to CII population analytics
    → Better prescribing intelligence returns to physician
    → [LOOP: each visit generates clinical + pharmacy revenue]
```

**Compound effect:** TAP Pharmacy adds margin to every clinic without additional patient acquisition cost. At 71 clinics, the dispensing revenue becomes substantial.

### Loop 2: Home Care → CDM → Outcomes Data

```
Integrative Health delivers home care
    → CDM program manages chronic disease patients
    → PROMs collected → flow to VBH Co
    → VBH Co proves intervention effectiveness
    → Payers agree to value-based contracts
    → More CDM patients enrolled
    → [LOOP: outcomes data proves value, funding flows back]
```

### Loop 3: Clinics → Data → AI → Better Care

```
Welunion clinics generate encounter data
    → CII ingests and trains clinical AI models
    → AI documentation, coding, and decision support deployed back
    → Physicians deliver more accurate, better-documented care
    → Integra RCM captures higher reimbursement from better docs
    → More revenue → more clinic expansion → more data
    → [LOOP: data makes AI smarter, AI makes clinics better]
```

### Loop 4: Clinical Ventures → Specialty Expansion

```
Clinical Ventures launches specialty clinic (mental health, vascular)
    → Specialty encounter data enriches population models
    → Evidence Platform adds specialty guidelines
    → Nama CRO recruits specialty patients for trials
    → VBH Co measures specialty outcomes
    → More specialty ventures attract more physicians
    → [LOOP: each specialty deepens ecosystem coverage]
```

## THE DATA SPONGE PRINCIPLE

Welunion is designed as a "data sponge" — it absorbs data from every patient interaction and releases structured intelligence to the rest of the ecosystem:

| Data Type | Volume (Y5) | Destination |
|-----------|-------------|-------------|
| Clinical encounters | 71 clinics × daily volume | CII, Integra RCM |
| Home care visits | 265,800/year | CII, VBH Co |
| CDM enrolled members | 10,250 | VBH Co, Nama CRO |
| Prescriptions | Every encounter | TAP Pharmacy, CII |
| PROMs responses | CDM + chronic patients | VBH Co |
