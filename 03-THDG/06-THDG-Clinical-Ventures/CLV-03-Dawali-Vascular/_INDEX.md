## Identity

**Dawali** (دوالي) is a specialized interventional radiology network currently operating **3 established branches** in Saudi Arabia, with a validated expansion plan to **13 branches nationally**. ## Ecosystem Integration — The Drawer Model

Dawali sits inside a **three-entity referral drawer** with Welunion and CoClinics. Services escalate and de-escalate seamlessly between the three layers:

```
┌─────────────────────────────────────────────────────────────┐
│                    PATIENT ENTRY POINTS                      │
└───────────┬─────────────────────┬───────────────────────────┘
            │                     │
            ▼                     ▼
   ┌─────────────────┐   ┌──────────────────┐
   │   WELUNION      │   │   COCLINICS       │
   │ (Primary Care + │   │ (Shared Specialty │
   │  CDM + Digital) │   │  Co-Clinic Model) │
   └────────┬────────┘   └────────┬─────────┘
            │  Detects vascular,  │  Specialist referral
            │  fibroid, BPH,      │  when co-clinic
            │  diabetic foot,     │  capacity reached
            │  pelvic symptoms    │
            └──────────┬──────────┘
                       │
                       ▼
            ┌──────────────────────┐
            │      DAWALI          │
            │  Interventional Rad  │
            │  Specialist          │
            │  Procedures          │
            └──────────┬───────────┘
                       │
          Post-procedure return flow:
          - Follow-up care → Welunion CDM
          - Wound care / medication → TAP Pharmacy
          - Chronic monitoring → Welunion digital
          - Outcomes data → TVG VBH
```

**Referral Flow Rules:**
1. **Welunion detects → refers to Dawali:** Primary care physician or digital triage identifies varicose veins, fibroids, BPH, or diabetic foot vascular risk → structured referral with clinical summary to Dawali
2. **CoClinics → Dawali:** Co-clinic specialist (e.g., obstetrics, urology) identifies fibroid/BPH case requiring embolization → direct specialist-to-specialist referral
3. **Dawali → Welunion (post-procedure):** After any Dawali procedure, patient returns to Welunion for 30/60/90-day follow-up, CDM enrollment, and medication management
4. **Dawali → TAP Pharmacy:** Dawali procedure patients receive same-day prescriptions dispensed through TAP Pharmacy embedded in Welunion/CoClinics locations
5. **All three → TVG VBH:** Procedure outcomes, symptom resolution scores, and readmission data flow to VBH Company for outcomes measurement and insurer reporting

## Strategic Value to Theeb Ecosystem

| Dimension | Value |
|-----------|-------|
| **Service complexity escalation** | Welunion handles primary care; Dawali handles the high-value specialist intervention — one patient, one ecosystem |
| **Revenue capture at procedure level** | Interventional radiology procedures are among the highest-margin ambulatory services in Saudi healthcare |

| **Asset-light MSO model** | Theeb earns MSO fees without equity concentration risk; physicians are motivated owners |
